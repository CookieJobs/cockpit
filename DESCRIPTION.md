# Cockpit — Project Description

## Short (1 行，repo 标题副标题 / About)

> Cockpit is a local-first personal agent that captures what you do, turns completed work into reusable achievements, and helps you write weekly reports and 述职 materials — powered by your own LLM API key, with your data staying on your own machine.

## 中文版（公众号 / V2EX / 知乎发布用）

> Cockpit（拾光）是一个本地优先的个人工作助理。它在你随手记录任务的时候，悄悄把"完成的事"沉淀成可复用的成就素材，周报、述职、CV 都不再翻聊天记录 — 接你自己的 LLM API key，数据完全留在你本机。

## 长版（README 顶图 / landing hero）

> Cockpit — Capture everything you do. Build the case for yourself.
>
> 还在每周 5 写周报？还在述职时翻一年聊天记录？Cockpit 把工作流的"完成"动作直接接上沉淀漏斗：完成任务 → 立刻生成 outcome / CV 描述 → 周报、述职、CV 模板一键导出。
>
> 不用服务器，不用绑定账号，不用订阅，5 分钟下载，本地数据库立等可用。

## Tech tagline（1 行）— for tweet / 标题

> Local-first personal agent · Claude / DeepSeek / MiniMax 自带 · SQLite 单文件 · 0 服务器

## 用户收益（3 条 bullet — for landing page）

- ✅ **完成即沉淀**：完成任务时让你写一句 outcome + 选 CV 描述，自动积累成可复用成就库
- ✅ **周报 / 述职一键出**：模板引擎 + LLM 起草，10 分钟搞定"本周做了什么"和"这一年的产出"
- ✅ **数据留在本机**：SQLite 单文件 + 本地优先架构，配上你自带的 LLM key，隐私 / 成本都你说了算
